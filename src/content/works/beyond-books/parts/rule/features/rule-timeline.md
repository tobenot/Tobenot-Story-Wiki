**鸿律篇时间线** 按正文时序梳理本篇事件。鸿律篇为单线第一人称、按真实时序推进，时间线与正文顺序基本一致；本页主要标注**作者世界回溯**与**闪回**（即苏敬峰借异能回到刻字场景、未来穿越者来历等非当下时序的片段），便于追更读者理清、也便于作者排查矛盾。

> 本页含全部剧情揭示。各条细节置于剧透块内，按需展开。⮕ 表示该信息在正文中后于此时点才被揭示。

## 时序备注

- 本篇全程苏敬峰第一人称，无多视角交错；时间线 ≈ 正文揭示顺序。
- 作者世界的回溯片段（如规律机构墙刻场景）发生在过去，但苏敬峰于当下进入，下文标注为「回溯」。
- 穿越者来自「几百年或几千年」之后，其原本时间线远在篇末之后，仅以「未来」标注。

## 一、序幕：时光机事件（第一章）

:::spoiler source="《不止于纸上的故事：鸿律篇》作者世界 / 时光机事件"
苏敬峰凭【作者世界】经招工杂志上的时光机论文进入光明境（回溯：见漂浮岛屿、双日），获研究中心实习。首日目睹人类首次时光机实验，未来穿越者自门框探出拔断电线中止实验、警告敌人可经通道而来。实验永久停止，苏敬峰失业。⮕ 穿越者来历（自愈、断身存活）于第二章「执念」由导师补述。
:::

## 二、闯入规律机构（第二章）

:::spoiler source="《不止于纸上的故事：鸿律篇》规律机构 / 无光空间"
苏敬峰为寻原稿，借地图文字穿梭各实验室，发现虚掩门内的秘密实验室，墙刻「规律机构」。探知墙刻时回溯至刻字场景，见行凶者与迎接者，二人对他态度迥异（一呼「敬峰哥」、一喊「杀了你」）。随后行凶者熄灯发起无光空间死斗，苏敬峰借作者世界回溯服务器结构引爆之，击杀行凶者，自身重伤昏迷。
:::

## 三、病床与初见银（第三章）

:::spoiler source="《不止于纸上的故事：鸿律篇》病床 / 无垠草地 / 搀扶"
苏敬峰病床醒来，穿越者远程自愈能力跟随。得知王烛馨腹痛住院，他赶往医院，途中被大理石短暂抓入无垠草地见一长袍人影（银，未深究）。等红灯时遇目盲者，银警告「不要让她碰到」。苏敬峰以莫名出现的逆模因手枪击退目盲者，目盲者被车撞飞。⮕ 手枪来历「来自光明境的礼物」于同节由说明书揭示。
:::

## 四、烛馨觉醒与逆模因伏击（第四章前半）

:::spoiler source="《不止于纸上的故事：鸿律篇》烛馨同学 / 沉重之色 / 勿忘我"
苏敬峰抵医院与烛馨重逢，发现花色食物异常、烛馨觉醒失控（吊扇松脱、连环车祸、高铁脱轨）。再入草地，银泪求「把她带过来」。携烛馨赴大理石途中遭逆模因伏击，苏敬峰自轰左眼入光明境，花蕊授反制图案，返回乱枪击杀刺客。穿越者驾车接二人至大理石。
:::

## 五、铜像之战与光明境真相（第四章后半）

:::spoiler source="《不止于纸上的故事：鸿律篇》铜像 / 大理石"
铜像男现身广场赠枪「公平对决」，雕像化死斗后自困雕像。战后烛馨演示复原铜像。银现身揭示光明境危机、授存在法则与课桌锚点，苏敬峰以灵力将烛馨拉入光明境。
:::

## 六、规律机构终战（第五章）

:::spoiler source="《不止于纸上的故事：鸿律篇》聚散之律 / 追忆 / 银剑 / 尾声"
苏敬峰携烛馨直捣规律机构。迎接者杀烛馨、消死亡概念、悟鸿律；银以银剑与图案救苏敬峰脱困。苏敬峰操场追忆，借课桌锚点再入光明境，烛馨悟鸿律复生。苏敬峰借墙刻将迎接者拉入光明境，烛馨持银剑斩之并「令其从未存在」。烛馨拆仪器成粉末，二人归还光明境能量、与银告别，去结婚。
:::
