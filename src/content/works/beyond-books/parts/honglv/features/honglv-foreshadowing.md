**鸿律篇伏笔与悬念追踪** 逐条记录本篇埋下的伏笔与悬念，标注**设悬处**、**当前状态**与**揭晓与去向**。状态分三类：

- **已揭晓**：本篇内已给出明确答案。
- **部分揭晓**：本篇内给出部分答案，仍有未解之处。
- **未揭晓**：本篇未解，指向后续篇章或留作悬念。

> 本页含全部剧情揭示。各条揭晓内容置于剧透块内，按需展开。

## 苏敬峰的「通道」资质

:::spoiler source="《不止于纸上的故事：鸿律篇》规律机构 / 铜像 / 大理石"
**状态：已揭晓。**

设悬：行凶者于墙刻回溯中一见苏敬峰便喊「杀了你」，似与他早有宿怨；苏敬峰不懂其故。

揭晓：铜像男与银先后说明——苏敬峰具「通道」资质，能进入别的宇宙或让别的宇宙进入他，故被规律机构招揽与清除。其异能【作者世界】即此资质的来源。
:::

## 逆模因手枪的来历

:::spoiler source="《不止于纸上的故事：鸿律篇》搀扶 / 彩光"
**状态：部分揭晓。**

设悬：赴医院途中，苏敬峰口袋里的水果刀莫名变成一把逆模因手枪，来源不明。

揭晓：说明书末页以未知语言写「来自光明境的礼物」。王烛馨后述该枪曾用于拯救银所保卫的[[location.heart-edge-realm|心棱域]]，「是它找到你的」。⮕ 谁让枪出现在他口袋、为何选中他，本篇未正面交代。**未揭晓。**
:::

## 目盲者与烛馨觉醒

:::spoiler source="《不止于纸上的故事：鸿律篇》搀扶 / 沉重之色"
**状态：部分揭晓。**

设悬：目盲者称苏敬峰「和她一样」；王烛馨突然腹痛觉醒，病因不明。

揭晓：苏敬峰判断目盲者曾触碰王烛馨，是烛馨腹痛觉醒的源头。⮕ 目盲者受谁指使（疑规律机构）、其触碰为何能诱发觉醒，本篇未明示。**未揭晓。**
:::

## 穿越者的身份与时代

:::spoiler source="《不止于纸上的故事：鸿律篇》时光机事件 / 执念 / 病床"
**状态：部分揭晓。**

设悬：穿越者自未来而来，自愈、断身存活，来历成谜。

揭晓：导师补述她来自几百年或几千年之后，因自愈异能被选来传话；她称「这个时代还有异能可以用，我在那个时代已经无法使用异能了」「有肉体的人也没多少个，能认识的基本都是机器」。⮕ 她所在未来的具体状况、与亿万年后敌人的关系，本篇未展开。**未揭晓。**
:::

## 鸿律

:::spoiler source="《不止于纸上的故事：鸿律篇》大理石 / 聚散之律 / 追忆 / 银剑"
**状态：已揭晓。**

设悬：篇名「鸿律」、银所言之存在法则、迎接者消除「死亡」概念，皆指向一个未明的底层规律。

揭晓：鸿律即「所有宇宙的规律」。领悟者其能量更标准、更合于律。王烛馨在光明境理解所有宇宙规律，能量胜过悟律的迎接者，持[[item.silver-sword|银剑]]将其斩杀并「令其从未存在」。银的存在法则（连续想象即存在）亦为鸿律的体现。
:::

## 光明境与心界的层级

:::spoiler source="《不止于纸上的故事：鸿律篇》大理石 / 彩光"
**状态：已揭晓。**

设悬：苏敬峰经论文进入的「漂浮岛屿、双日」世界是什么；银自称心界人，与光明境何关。

揭晓：该世界即[[location.bright-realm|光明境]]，心界是其子空间（[[location.heart-realm|心界]] ⊃ [[location.heart-edge-realm|心棱域]]），银为心界圣人、心棱域守护者。光明境另有其他空间，本篇未展开。
:::

## 迎接者的不死

:::spoiler source="《不止于纸上的故事：鸿律篇》聚散之律 / 银剑"
**状态：已揭晓。**

设悬：迎接者被粉碎成尘土仍能复活，自述「消除了我身上死亡的概念」，似不可杀。

揭晓：王烛馨以更高标准的鸿律能量胜之，未直接「杀死」他，而是「让他从来没有存在过」，绕过其不死性。
:::

## 行凶者与迎接者对苏敬峰的预知

:::spoiler source="《不止于纸上的故事：鸿律篇》规律机构"
**状态：未揭晓。**

设悬：墙刻回溯中，行凶者与迎接者刻字时便「认识」苏敬峰——一呼「敬峰哥」、一喊「杀了你」，且苏敬峰的「招人名单」早有他二人。

去向：本篇未解释他们如何预知苏敬峰、为何一友一敌。可能与「通道」资质的跨时空性质、或机构对苏敬峰的长期布局有关，留作后续篇章悬念。**未揭晓。**
:::
