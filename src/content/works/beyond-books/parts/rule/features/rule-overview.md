**《不止于纸上的故事：鸿律篇》** 是《不止于纸上的故事》系列的一个篇章，写于 2023 年 2 月，作者苏敬峰，献给王烛馨。本篇为单线第一人称叙事，全程跟随视角人物苏敬峰，围绕光明境的能量危机与规律机构的阴谋，苏敬峰与恋人王烛馨在银、花蕊等心界人的引导下，逐步触及所有宇宙的底层规律——鸿律。

本页是鸿律篇的「门面」，提供无剧透的骨架速览与章节结构说明，并通往 [[feature.rule-timeline|时间线]]、[[feature.rule-foreshadowing|伏笔与悬念追踪]] 等工具页。

## 核心冲突

苏敬峰因异能【作者世界】被[[location.rule-institution|规律机构]]视为能沟通宇宙的「通道」，先被招揽、后遭清除。机构暗藏于[[location.time-machine-center|时光机研究中心]]内，以仪器汲取[[location.bright-realm|光明境]]的能量、毁灭心界。苏敬峰的恋人王烛馨因与他灵力相通，被光明境能量选中，能力随光明境消亡而不断增强，成为破局关键。神器可能是假的，记忆可能是假的，连「死亡」都能被消除——直到触及万物底层之「鸿律」。

## 章节结构与叙事视角

与银月篇的多视角非线性剪辑不同，鸿律篇为**单线第一人称**，全程由苏敬峰叙述，按真实时序推进，无主线交错。正文五章：

- **第一章**（作者世界 / 时光机事件）：苏敬峰的异能与来历，时光机实验被中止，首日失业。
- **第二章**（执念 / 规律机构 / 无光空间）：为寻论文原稿闯入规律机构，与行凶者无光空间死斗。
- **第三章**（病床 / 无垠草地 / 搀扶）：穿越者庇护、初见银、目盲者遭遇。
- **第四章**（烛馨同学 / 沉重之色 / 勿忘我 / 彩光 / 铜像 / 大理石）：烛馨觉醒失控、逆模因伏击、铜像之战、银揭示光明境真相。
- **第五章**（聚散之律 / 追忆 / 银剑 / 尾声）：终战规律机构、烛馨悟鸿律复生、银剑斩迎接者、归还能量。

## 登场角色一览

> 以下按公开身份列出，命运转折与真实身份属于剧透，见各角色页与伏笔追踪页。

### 主角方

- [[character.su-jingfeng|苏敬峰]]：视角人物，大三学生，异能【作者世界】，被机构视为「通道」。
- [[character.wang-zhuxin|王烛馨]]：苏敬峰的恋人，觉醒结构操控异能。
- [[character.traveler|穿越者]]：来自未来的女子，远程自愈能力跟随苏敬峰。
- [[character.silver|银]]：心界圣人、心棱域守护者，引导者。
- [[character.hua-rui|花蕊]]：心棱域先遣天使队队长，于光明境接教苏敬峰。

### 规律机构

- [[character.welcomer|迎接者]]：核心执行者，悟鸿律、消死亡概念求不死，终战反派。
- [[character.assailant|行凶者]]：刻字者，无光空间死斗对手。
- [[character.statue-assassin|铜像男]]：派来的末位刺客，雕像化自愈。
- [[character.blind-one|目盲者]]：触碰型刺客，疑为烛馨觉醒诱因。

## 核心事件脉络

> 仅交代背景与性质，结果见各事件页与时间线页。

1. [[event.time-machine-incident|时光机事件]] — 人类首启时光机，被穿越者中止。
2. [[event.lightless-space-battle|无光空间死斗]] — 苏敬峰闯规律机构，首战行凶者。

7. [[event.bright-realm-revelation|光明境真相]] — 银揭示本篇核心真相、授往返光明境之法。
