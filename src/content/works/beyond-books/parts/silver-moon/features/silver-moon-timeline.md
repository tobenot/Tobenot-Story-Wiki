**银月篇时间线** 按**真实发生顺序**还原本篇事件，与《导演剪辑版》正文的非线性剪辑相对。每条标注「正文揭示位置」——即该事件在正文哪条主线、哪一节被揭示，便于追更读者理清时序、也便于作者排查矛盾。

> 本页含全部剧情揭示。各条细节置于剧透块内，按需展开。若只想看骨架，可只读各阶段标题与事件名。

## 时序备注

- 《导演剪辑版》正文称 [[character.tanaka-ren|田中连]] 于 2015 年上任 [[location.baylis|贝利斯]] 并引发本篇冲突；旧有时间线条目作 2011 年。二者尚未统一，以正文为准待校正。
- 正文的「主线」编号（罗伯特主线 1–4、贝利斯主线 1–4、杰瑞斯主线 1–2）是**揭示顺序**，不是发生顺序。例如杰瑞斯主线 2 揭示的「偷袭史蒂文」实际发生在破晓行动同期，而非正文出现的位置。
- 下文「⮕」表示该条在正文中后于此时点才被揭示（即典型的悬念前置）。

## 一、篇前：布局与渗透

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线1 / 罗伯特主线1 闪回"
田中连上任贝利斯，向 [[concept.sin-hunters|罪恶猎手]]「借取」其收缴的能量之体 [[item.oblivion|遗忘]] 与 [[item.greed|贪婪]]，着手研究。此时飞雁市多有异能者暴动，贝利斯大部队在市区镇压。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 闪回"
[[character.steve|史蒂文]] 接任银月寺主持，扩大招人、上书将保留地边界扩至围墙外，举行上任仪式给 [[character.silver-moon|银月]] 烧香。管理一度混乱。
:::

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线1"
田中连觊觎银月寺 [[item.refraction|折射]]，安排间谍对 [[character.caspar|卡斯帕]] 的车做手脚——卡斯帕可能是下任主持，预先削弱其防御工程能力。
:::

:::spoiler source="《不止于纸上的故事：银月篇》杰瑞斯主线1"
[[character.jerris|杰瑞斯]] 散步至 [[location.autumn-gaze-lake|望穿秋水]]，向 [[character.joan|琼]] 买下一块有灵力的伴生矿石（非能量之体），用以制作 [[item.greed|贪婪]] 的赝品。此事发生在罗伯特首次到访望穿秋水之前。⮕ 杰瑞斯主线 2 才揭示这块石头的用途。
:::

:::spoiler source="《不止于纸上的故事：银月篇》杰瑞斯主线2 杰瑞斯偷盗"
杰瑞斯用买的矿石嵌入自制的【贪婪】模型，制成有微弱灵力的赝品。他隐身潜入罪恶猎手仓库，趁元帅向贝利斯运送战利品之际，在货车内将真【贪婪】（黄电光、白底金纹匕首）与赝品调包。真品藏于胸前，另用一块矿石碎片掩盖残留灵力。⮕ 此后田中连佩戴的【贪婪】实为赝品。
:::

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线2 珂茵的恋人"
[[character.corn|珂茵]] 是贝利斯高级军官，卧底银月寺，与 [[character.clement|克莱门特]] 是恋人，每周密会。克莱门特向她展示二人在飞雁市救出的孤儿照片，主张「要有力量，保护更多人」。
:::

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线2"
田中连从珂茵处获知银月寺「警示作战」情报，军师建议直接进攻；田中连顾虑舆论，犹豫是否先发制人。
:::

## 二、晨曦：导火索

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 晨曦救援"
[[event.dawn-rescue|晨曦救援]]。罗伯特上班途中救下路中央的三花猫与失控车内的卡斯帕，发现刹车油管断口整齐、地面有未干刹车油——系人为谋杀。⮕ 贝利斯主线 1 揭示主使为田中连安排的间谍。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 贝利斯使者来索要折射"
田中连带使者与 [[character.airin|艾琳]] 上门索要【折射】，使者以艾琳相威胁，被罗伯特用时停顶飞，并被行车记录仪拍下。田中连递名片招揽被拒。⮕ 这段录像成为「暴力抗法」的借口。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1"
罗伯特向卡斯帕报告，卡斯帕转告史蒂文。史蒂文聚焦于贝利斯的侵犯，让大家准备「警示作战」远征。
:::

## 三、破晓行动

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线1 / 贝利斯主线2"
田中连以「罗伯特攻击使者」「银月寺不愿给折射」为由发动 [[event.dawn-break-operation|破晓行动]]，突击队含少量人类与三名异能者雇佣兵 [[character.tours|图尔斯]]、[[character.stuart|斯图尔特]]、[[character.clement|克莱门特]]。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 破晓行动 / 贝利斯主线2"
战斗分头进行：克莱门特对珂茵、斯图尔特对诺顿、图尔斯对罗伯特（南门单挑）。珂茵故意放空箭，让克莱门特飞入银月寺用特殊炸药炸开地下室门，将【折射】装入能掩盖灵力的特殊箱子抢走。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1"
【折射】被装箱后环境灵力骤减，罗伯特时停失效，图尔斯生成的钢铁消失致其被重创，战斗中止。贝利斯三人往西北飞行撤退。⮕ 此时只有罗伯特视角，珂茵失守、辛妮亚重伤待通知。
:::

:::spoiler source="《不止于纸上的故事：银月篇》杰瑞斯主线2 「破晓行动」中罪恶猎手的行动"
**与破晓行动同期。** 杰瑞斯趁乱隐身潜入银月寺：偷袭独自苦战的 [[character.xinnia|辛妮亚]]，用真【贪婪】刺其腹部夺取【音域净空】；又潜入主持房间刺伤史蒂文，夺取【钢铁之躯】（卡斯帕默许此打击）。辛妮亚与史蒂文均以为是贝利斯所为——嫁祸成立。⮕ 罗伯特主线 2 才以「消除异能武器」谜面呈现，杰瑞斯主线 2 才揭示真相。
:::

:::spoiler source="《不止于纸上的故事：银月篇》杰瑞斯主线2 双重虚假"
杰瑞斯潜入时发现真【折射】已不在原位、状若「刚拿走」，便顺手放一个只有光泽的假【折射】于地下室，令贝利斯抢到空壳。⮕ 真【折射】去向成谜，留至彩蛋。
:::

## 四、森林追击与冰雪风暴

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 森林追击"
史蒂文公告【折射】被偷，派罗伯特、[[character.norton|诺顿]] 追击；辛妮亚重伤下场，珂茵待恢复后守卫。诺顿开车接罗伯特，电话中珂茵告知辛妮亚重伤。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1"
途中罗伯特再见那只三花猫，猫朝他摇头。⮕ 未揭晓，疑为伏笔。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1"
贝利斯雇佣兵在森林遇一队罪恶猎手发生战斗，罪恶猎手败退，反将雇佣兵位置情报报给银月寺二人。罗伯特、诺顿在河边追上正检查安全箱（【折射】）的三名雇佣兵。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 冰雪风暴"
琼从森林走出，认出斯图尔特，决定相助，释放冰雪风暴，环境灵力再降一个数量级，众人异能盾布满裂痕。罗伯特打飞图尔斯、击落斯图尔特，夺回【折射】交诺顿带回。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 / 贝利斯主线2 瞬移"
罗伯特被三人围攻、琼一旁射冰锥，陷入绝境。珂茵瞬移至场，告知「史蒂文被重伤」，逼退三人，带罗伯特一起瞬移离开。战场只剩贝利斯三人与琼，斯图尔特约琼下次。⮕ 珂茵「两个世界的边缘」的两难，贝利斯主线 2 才揭示。
:::

## 五、卧榻之侧：调查与疑云

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2 卧榻之侧"
罗伯特探望辛妮亚——她已失去异能，将回归人类社会。又探望史蒂文——同样失去异能（钻石皮肤失效）、自划伤口，亦将离开银月寺。二人腹部均有捅刺伤口，均称只记得一把带黄色电光的白底金纹匕首飞插进来。罗伯特怀疑贝利斯有「消除异能的武器」或隐形刺客，半夜勘察遇袭地点，无大规模破坏痕迹。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2"
罗伯特暗想：【折射】既已拿回、又不在琼的冰雪风暴影响下，为何银月寺环境灵力仍然稀少？——低灵力之谜，未完全解答。
:::

## 六、贝利斯主线 3：假货、探测器与渗透

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线3 贝利斯的愤怒"
贝利斯撤退到望穿秋水附近扎营检查【折射】，发现是假货，田中连大怒，遂策划后续「清扫行动」。同时田中连从罪恶猎手得到【贪婪】【遗忘】（其中【贪婪】已是杰瑞斯调包后的赝品），研究后做可探测周边十米能量之体的探测器。田中连将【贪婪】佩戴腰间，测试可防狙击枪子弹（实为赝品的微弱表现），非常自信。
:::

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线3 两个世界的边缘"
珂茵向田中连抱怨辛妮亚重伤，被斥「心软」。田中连派她破坏北门防御，她接下却不打算全数执行。
:::

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线3 配合贝利斯"
珂茵晚上开会，将弩机等防御布置泄露给贝利斯。⮕ 此为清扫行动中城墙弩机被毁的伏笔。
:::

## 七、建设期：高筑墙、广积粮

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2 卡斯帕的建设 / 躬耕银月"
卡斯帕上位后改取「高筑墙，广积粮」，组织城墙弩机（朝内安装）、农田、风力发电机等建设。罗伯特、诺顿、珂茵报名项目，罗伯特修弩机、学操作，期间反复向间谍追问「消除异能武器」无果。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2 棍术课堂 / 夜间巡逻"
罗伯特在银月寺高中大学宣讲，卡斯帕教棍术、演示灵能盾、讲解异能溢出。夜间巡逻中罗伯特处理破坏弩机的突发事件，并在巡逻交界处等珂茵未果，诺顿前来调侃。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2 周日的外出"
休息日罗伯特出西门遇罪恶猎手四人（冷锋、银狐、铁拳、狂花），被其合法持枪挑衅。罗伯特逃入森林，森林骤起诡异寒风与冰锥专击罪恶猎手——琼暗中出手驱逐。罗伯特不知所以。
:::

## 八、望穿秋水线

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2 第一次进入望穿秋水"
罗伯特带礼物到望穿秋水，向琼说明贝利斯抢【折射】的真相。琼无兴趣，以攻击代替交谈切磋，最终以一道绚丽冰雕收手赶罗伯特走。
:::

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线3 望穿秋水的冲突"
田中连监测到望穿秋水高能反应（琼与罗伯特战斗、或【星辰】波动），派图尔斯、克莱门特前往。珂茵泄露罗伯特外出情报，本意引贝利斯抓罗伯特，却引出【星辰】。⮕ 罗伯特视角不知自己被出卖。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2 望穿秋水会战"
[[event.autumn-gaze-battle|望穿秋水会战]]。图尔斯、克莱门特奉命征湖底 [[item.stardust|星辰]]，琼冰封反击。克莱门特扎入湖水，【星辰】主动飞向其手——他成为「选中者」，羽翼骤然强化，与图尔斯逃走。琼震惊道出「选中者」。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2"
罗伯特泡湖恢复灵力，在意识中见到一条冰晶构成的冰龙。琼称自己也见过同样的冰龙，并记入笔记。琼决意不再站贝利斯，打电话让斯图尔特离开贝利斯来见她。⮕ 冰龙之谜未解，悬念延续。
:::

## 九、清扫行动：最终决战

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线4 / 罗伯特主线3"
[[event.sweep-operation|清扫行动]]。贝利斯四门同时进攻：斯图尔特南门、克莱门特东门、图尔斯东门缓推；诺顿守北门、珂茵守西门、罗伯特守南门，卡斯帕对讲机指挥。贝利斯间谍已据珂茵泄露的情报摧毁城墙弩机。图尔斯等以【星辰】为核心制作新武器。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线3 决战！银月寺游击战"
斯图尔特南门放水撤退；罗伯特驱车截东门突入的克莱门特，珂茵光箭击落其羽翼，大学生弩机支援，克莱门特撤退。间谍摧毁所有城墙弩机后，珂茵自行去北门支援诺顿。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线3 / 杰瑞斯主线2"
东门罗伯特对图尔斯。举枪队员接连倒下、子弹打中空气被弹飞——杰瑞斯隐身现身，自称罪恶猎手高官，表示罪恶猎手站银月寺一方，随后进入寺内支援其他战场。罗伯特击退图尔斯，图尔斯称不知消除异能武器。
:::

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线3 应弦而倒"
城墙学生修好唯一未被大破坏的巨型弩机（此前珂茵守过此处）。田中连从银月大学张扬走出，二楼冷箭被其腰间【贪婪】赝品的电光弹飞。罗伯特以【苍穹缝隙】校正风偏，弩箭穿过赝品防御命中田中连胸口，重伤。贝利斯护送其撤退。⮕ 田中连自恃的「神器」实为赝品，杰瑞斯主线 2 冷笑揭示。
:::

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线4"
珂茵自伤假装受伤，安排渗透者进行暴露身份的偷袭。贝利斯搜刮科技文件与有用资料（战斗主要目的）。田中连醒来后求和，此后一直住院。
:::

## 十、尾声与彩蛋

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线4 篝火烤鱼"
战后罗伯特探望珂茵、诺顿。贝利斯送来道歉书与大量白银赔偿，承诺不再征取【折射】。田中连重伤生死未卜，银月寺暂安。
:::

:::spoiler source="《不止于纸上的故事：银月篇》彩蛋-珂茵的调查"
