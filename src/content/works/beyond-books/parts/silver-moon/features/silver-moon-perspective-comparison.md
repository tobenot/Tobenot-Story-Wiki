**银月篇多视角事件对照** 选取本篇中由多条主线分别揭示的关键事件，并排呈现各视角已知的信息，标示「读者在何时知道多少」。本篇采用「希区柯克式」非线性剪辑，同一事件的结果、动机、真相往往分属罗伯特线、贝利斯线、杰瑞斯线，拼合后才得全貌。

> 本页含全部剧情揭示。各视角信息置于剧透块内，按需展开。

## 剪辑手法与信息差

正文的四条主线并非平行叙事，而是**信息分层揭示**：罗伯特线最先给出「结果与谜面」，贝利斯线补足「动机与误判」，杰瑞斯线最后揭开「真相」。读者跟随罗伯特产生疑问，又比罗伯特更早（在贝利斯线、杰瑞斯线中）获知答案——这种「读者比角色多知道一些」的落差，正是悬念的来源。下文每个事件给出各视角的揭示内容，并注明读者在哪条主线首次获知关键真相。

## 卡斯帕车祸

### 罗伯特视角

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 晨曦救援"
罗伯特救下三花猫与卡斯帕，发现刹车油管断口整齐、地面有未干刹车油，断定为谋杀；但疑惑卡斯帕身为棍术高手不应失控得如此彻底。**不知主使是谁。**
:::

### 贝利斯视角

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线1"
田中连安排间谍对卡斯帕的车做手脚，因卡斯帕可能是下任主持、欲防其多做防御工程。
:::

**读者何时知道多少**：罗伯特线只给谜面；贝利斯线首次揭示主使与动机。

## 破晓行动

### 罗伯特视角

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 破晓行动"
罗伯特只见战斗本身：分头迎战，[[item.refraction|折射]] 被装入掩灵箱子抢走，环境灵力骤减致其时停失效、[[character.tours|图尔斯]] 被重创，战斗中止；珂茵那边「失守、待通知」，辛妮亚重伤。**以为是贝利斯正面入侵，不知内应，不知「消除异能武器」真凶。**
:::

### 贝利斯视角

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线2 破晓行动的准备 / 配合"
田中连从 [[character.corn|珂茵]] 处获知「警示作战」情报并决定先发制人；又从珂茵处得知【折射】所在地道，令珂茵故意放空箭、让 [[character.clement|克莱门特]] 飞进去抢走。**揭示珂茵是内应、行动是配合好的。**
:::

### 杰瑞斯视角

:::spoiler source="《不止于纸上的故事：银月篇》杰瑞斯主线2 「破晓行动」中罪恶猎手的行动 / 双重虚假"
杰瑞斯趁乱潜入，偷袭辛妮亚、史蒂文夺异能并嫁祸贝利斯；发现真【折射】已不在原位，顺手放假【折射】让贝利斯抢空壳。**揭示「消除异能武器」真凶是杰瑞斯，且真【折射】早被他人取走。**
:::

**读者何时知道多少**：罗伯特线只看到表层战果与谜面；贝利斯线揭示内应与配合；杰瑞斯线最后揭开「消除异能武器」真相与「假折射」。

## 森林追击与冰雪风暴

### 罗伯特视角

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线1 森林追击 / 冰雪风暴"
罗伯特追击、琼介入耗尽灵力、夺回【折射】交诺顿带回，自己被围困；珂茵瞬移至场告知「史蒂文被重伤」，带他离开。**不知琼为何帮贝利斯、不知珂茵为何能瞬移救他、不知珂茵与克莱门特的关系。**
:::

### 贝利斯视角

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线2 瞬移"
珂茵看着场上克莱门特等人与满身是伤的罗伯特，迟疑许久，率先拉弓用光箭逼退三人，抓住罗伯特一起瞬移离开。**揭示珂茵「两个世界的边缘」的两难——她不忍罗伯特被杀。** 战场只剩贝利斯三人与琼。
:::

**读者何时知道多少**：罗伯特线呈现「珂茵突然救场」的结果；贝利斯线揭示其内心矛盾与动机。

## 望穿秋水会战

### 罗伯特视角

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线2 望穿秋水会战"
图尔斯、克莱门特来征 [[item.stardust|星辰]]，琼冰封反击，克莱门特被【星辰】「选中」、羽翼强化逃走；罗伯特困惑「选中者」机制，并见到冰龙。**不知贝利斯为何恰好出现在此。**
:::

### 贝利斯视角

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线3 望穿秋水的冲突"
田中连监测到望穿秋水高能反应，派图尔斯、克莱门特前往；珂茵泄露罗伯特外出情报，本意引贝利斯抓罗伯特，却意外引出【星辰】；克莱门特获【星辰】认可对贝利斯是意外之喜。**揭示罗伯特是被珂茵出卖才在此遇袭。**
:::

**读者何时知道多少**：罗伯特线只见战斗与「选中者」；贝利斯线揭示袭击的真正起因是珂茵的出卖。

## 清扫行动 · 应弦而倒

### 罗伯特视角

:::spoiler source="《不止于纸上的故事：银月篇》罗伯特主线3 决战 / 应弦而倒"
四门防守，东门「黑影」杰瑞斯现身清兵、自称罪恶猎手高官；城墙学生修好唯一巨型弩机，罗伯特以【苍穹缝隙】校正风偏，弩箭命中张扬走出的田中连胸口。罗伯特透过瞄准镜见其腰间白底金纹匕首，断定那就是「消除异能的武器」。**罗伯特以为真相大白，实则只看到赝品。**
:::

### 贝利斯视角

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线4 最终决战"
珂茵泄露防御图致城墙弩机被毁、自伤假装受伤、安排渗透者暴露身份偷袭；贝利斯搜刮科技资料（主要目的）。田中连自恃腰间「神器」【贪婪】、以为所有弩机均已瘫痪，大摇大摆走出掩体，被唯一修好的弩机射趴。醒来后求和。
:::

### 杰瑞斯视角

:::spoiler source="《不止于纸上的故事：银月篇》杰瑞斯主线2 【清扫行动】中的罪恶猎手"
杰瑞斯隐身进入战场，在东门帮罗伯特清理贝利斯士兵（即罗伯特看到的「黑影」）；当罗伯特射倒田中连、发现其腰间匕首时，杰瑞斯在暗处冷笑——那是他留下的赝品，足以作贝利斯「非法使用违禁武器」的铁证，而真正的力量（异能夺取）此刻正握在杰瑞斯自己手中。
:::

**读者何时知道多少**：罗伯特线让读者以为「消除异能武器」谜底是田中连的匕首；贝利斯线揭示田中连的误判与败因；杰瑞斯线最后反转——匕首是赝品，真凶与真品都在杰瑞斯处。三视角拼合方得全貌。

## 【折射】真伪之线

### 贝利斯视角

:::spoiler source="《不止于纸上的故事：银月篇》贝利斯主线3 贝利斯的愤怒"
贝利斯扎营检查抢来的【折射】，发现是假货，田中连大怒，遂策划清扫行动。**只知「被银月寺耍了」，不知真品何在。**
:::

### 杰瑞斯视角

:::spoiler source="《不止于纸上的故事：银月篇》杰瑞斯主线2 双重虚假"
杰瑞斯潜入时发现真【折射】已不在原位、状若「刚拿走」，便放一个假【折射】让贝利斯抢空壳。**揭示假货出自杰瑞斯之手，但真品去向他也不知。**
:::

### 彩蛋（珂茵视角）

:::spoiler source="《不止于纸上的故事：银月篇》彩蛋-珂茵的调查"
珂茵在船上念「真正的【折射】……」并望向 [[location.dance-dragon-mountain|舞龙山]]。**指向真品去向，留待后续。**
:::

**读者何时知道多少**：贝利斯线给「假货」谜面；杰瑞斯线揭示假货来源与「真品已先行消失」；彩蛋暗示去向但未坐实。
