**苏敬峰**是《不止于纸上的故事：鸿律篇》的核心视角人物，大三学生，持有异能【作者世界】。本篇全程以其第一人称展开。本页记载他在鸿律篇中的具体经历与该篇章揭示的关于他的信息；其跨篇章设定见全局条目·苏敬峰（待建）。

## 身份卡

- **所属篇章**：《不止于纸上的故事：鸿律篇》
- **身份**：大三学生、时光机研究中心实习生
- **异能**：【作者世界】（以文字为入口进入作者世界）
- **关键章节**：第一至五章（全程）

## 本篇定位

苏敬峰是鸿律篇唯一的视角人物，整篇由他的眼睛展开。一个为跟恋人王烛馨结婚而四处谋职的大三学生，凭【作者世界】这手「把文字读进去就能进去」的异能，误打误撞进了时光机研究中心实习——然后第一天就失业了。此后他被一股力量不断推着走，从求职者一步步被逼向与一个能操纵概念的存在正面相抗的境地。具体的转折与走向详见下文剧透块。

## 经历

### 首日实习

凭【作者世界】从招工杂志上一篇时光机论文进入光明境，又凭这趟异能体验换来研究中心的实习。报到首日，导师把他领到时光机所在的门框前，令他退到五米开外，宣告自己将回到五分钟前。研究人员齐按按钮，门框亮起白光。

:::spoiler source="《不止于纸上的故事：鸿律篇》作者世界 / 时光机事件"
光中伸出一只手，一名女子自门框探出半身、拔断电线中止实验，落地时断了一半身体。她警告众人不要启动时光机：「即使是亿万年后的敌人，也可以通过这个通道过来」，并称他们「毁掉了一整个别的宇宙」才抢在敌人之前抵达此时间点传话。实验自此永久停止，苏敬峰第一天即失业。
:::

### 循图探秘

为寻论文原稿，苏敬峰借导师给的地图穿梭各实验室，发现虚掩门内的秘密实验室，墙刻宇宙通用语「规律机构」。他探知墙刻时进入作者世界，见到两名刻字者。

:::spoiler source="《不止于纸上的故事：鸿律篇》规律机构"
其中一人见他大呼「敬峰哥」，另一人指着他喊「杀了你」——他们早已「认识」他。
:::

### 黑暗实验室

苏敬峰循线索摸进黑暗的实验室，行凶者于黑暗中以刻刀、美工刀偷袭，他重伤。

:::spoiler source="《不止于纸上的故事：鸿律篇》无光空间"
苏敬峰借咖啡烫敌、手电照明，并以作者世界回溯打火机与服务器的位置结构，击穿服务器外壳令电线短路爆炸，碎片扎满敌人身体，将其击倒。他重伤昏迷前见一束光自门口照入。
:::

### 病床

苏敬峰在病床醒来，伤口处长出豌豆大小的「小手」——穿越者的远程自愈能力正跟着他。穿越者告知他只昏迷了几小时，并称其误入的是规律机构而非档案室。他决意回学校，但得知王烛馨腹痛住院，立即赶往医院。

### 赴医院途中

赴医院途中，一位黄西装、眼珠浑浊的「目盲者」请他带去医院，苏敬峰退避时一度坠入草地、见银发女孩警告。

:::spoiler source="《不止于纸上的故事：鸿律篇》搀扶"
银发女孩警告「不要让她碰到，你会死的」。目盲者搭上他肩、令其腹部内容物被扭曲，苏敬峰摸出并非水果刀而是一把枪，凭说明书（作者世界）解除保险连击目盲者，恰被快车撞飞。他得知此枪为逆模因武器，末页写「来自光明境的礼物」。
:::

### 医院重逢

苏敬峰在医院与王烛馨重逢，发现病房花朵、烛馨盘中食物颜色异常，烛馨指甲咬痕透出异能盾蓝光，吊扇松脱、外界车祸、高铁脱轨——周遭一连串怪象同时发生。

:::spoiler source="《不止于纸上的故事：鸿律篇》烛馨同学 / 沉重之色"
苏敬峰判断是烛馨新能力失控所致，她已觉醒却无感觉。他借大理石照片进入无垠草地，银发女孩银流泪要求他「把她带过来这里」。
:::

### 赴大理石途中

携烛馨赴大理石途中，全城行人消失、记忆错乱、消息发不出去，烛馨被无形攻击割出多处伤口——敌方以逆模因令众人动作互不可见。

:::spoiler source="《不止于纸上的故事：鸿律篇》勿忘我"
苏敬峰以枪自轰左眼，进入光明境见到花蕊，习得反制逆模因的更强图案。返回后识破架在颈上的电锯，烛馨以能力拆解电锯，苏敬峰乱枪将刺客上身打空。
:::

### 广场

穿越者驾车接二人至大理石。规律机构派来的「铜像男」现身，称苏敬峰在招人名单上、具「通道」资质，赠突击步枪「公平对决」。

:::spoiler source="《不止于纸上的故事：鸿律篇》彩光 / 铜像"
铜像男的异能可化作白色雕塑自愈。苏敬峰以调包弹夹、油箱纵火等手段周旋，最终铜像男把自己完全变成雕像而无法逆转，被拆成碎石。
:::

### 大理石前

银现身，告知苏敬峰：光明境（他们的宇宙）正被规律机构毁灭。

:::spoiler source="《不止于纸上的故事：鸿律篇》大理石"
王烛馨因与光明境能量相通（穿过心口的七彩螺旋线）而能力会不断增强。苏敬峰立誓「哪怕对他们生命有一丝威胁，都要把他们从这世界抹去」。银授其「连续想象同一处即存在」的法则，令其在草地上具象课桌作为锚点，并要他「把王烛馨也拉进来」。
:::

### 聚散之律

苏敬峰携烛馨进入规律机构，击倒行凶者，与迎接者照面。

:::spoiler source="《不止于纸上的故事：鸿律篇》聚散之律"
迎接者以「后三指」令烛馨异能盾碎裂、又令其爆开染红长桌。苏敬峰取「最危险的试剂」欲砸迎接者，反被折手将试管刺入自己喉咙，承受迎接者消除「死亡」概念之力、周身碎裂。危急时银在光明境挥剑入其胸口、画白色亮线绘三图案，消去其「死亡」概念，他得以逃脱。
:::

### 追忆

苏敬峰独处操场，凭课桌锚点再入光明境，渴望再见烛馨，想象她跑来的样子。

:::spoiler source="《不止于纸上的故事：鸿律篇》追忆"
烛馨未实体出现，却以飘浮空中、白发彩衣、七彩线穿身之姿发声——她已理解所有宇宙的规律（鸿律），能量比迎接者更标准。她请他回去、让穿越者摸她的血以加强自愈。
:::

### 终战

苏敬峰返回现实，再度与迎接者交锋。

:::spoiler source="《不止于纸上的故事：鸿律篇》银剑"
苏敬峰借七彩线送土入规律机构，打迎接者一巴掌见其骨刺外露、并非异能者。迎接者悟得鸿律自愈、将其砸入服务器。苏敬峰借墙刻「规律机构」进入作者世界，将迎接者拉入光明境。王烛馨持银剑飞至，斩断迎接者白剑、将其砍成两半；血迹聚复为她的身体。她说「他死不了，但我让他从来没有存在过」。
:::

### 尾声

烛馨拆出仪器上刻名的金属，苏敬峰借其名进入作者世界。

:::spoiler source="《不止于纸上的故事：鸿律篇》尾声"
苏敬峰停掉仪器，烛馨将仪器拆成粉末。二人再入光明境归还能量，告别银。苏敬峰带烛馨去结婚，最后补吃了番茄汤麻辣烫，从一篮随机号牌里拿到了连着的号。
:::

## 该篇章的关系动态

### 王烛馨

苏敬峰的全部动机是「和王烛馨结婚」，两人以「敬峰哥/烛馨妹妹」相称。烛馨因他的能力被牵连。

:::spoiler source="《不止于纸上的故事：鸿律篇》烛馨同学 / 聚散之律 / 银剑 / 尾声"
烛馨觉醒、失控、一度被迎接者爆开消亡，他以「把她救回」为执念贯穿后半篇，最终在光明境重逢并取胜，携她去结婚。
:::

### 穿越者

穿越者的远程自愈能力持续跟着苏敬峰，多次救其性命，是苏敬峰信赖的救助者。

:::spoiler source="《不止于纸上的故事：鸿律篇》病床 / 彩光"
断指接回、心脏修复、伤口填满小手，皆赖其远程自愈；她在研究中心人员消失后开车接二人至大理石。
:::

### 银

银是苏敬峰的引导者：先警告他避开目盲者，后揭示光明境危机与「通道」资质，授其存在法则。

:::spoiler source="《不止于纸上的故事：鸿律篇》沉重之色 / 大理石 / 银剑"
最终在危急时以图案之力消去他「死亡」的概念。苏敬峰对她由困惑转为信任。
:::

### 迎接者

迎接者是规律机构核心执行者，本篇最终对手。

:::spoiler source="《不止于纸上的故事：鸿律篇》聚散之律 / 银剑"
他令烛馨爆开、对苏敬峰施「死亡」概念。苏敬峰对其由「不太坚定」转为「一定要把他从这世界抹除」，最终（借烛馨之力）令其从未存在过。
:::
